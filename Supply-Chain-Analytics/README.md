# Supply Chain Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes supply chain performance, sales trends, profitability, logistics efficiency, and regional performance.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin
- Total Orders

## Dashboard Pages
1. Overview
2. Logistics Analysis
3. Product Analysis
4. Regional Analysis

## Business Insights
- Identified top-performing regions
- Analyzed shipping mode performance
- Tracked product profitability
- Evaluated regional sales contribution

- Data Cleaning
- Data Modeling
- DAX Measures
- Dashboard Design
