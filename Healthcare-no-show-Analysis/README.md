# Healthcare No Show Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes patient appointment attendance patterns and identifies factors contributing to no-show behavior.

## Tools Used

- Power BI
- Power Query
- DAX
- Excel

## Key KPIs

- Total Appointments
- No Show Rate
- Show Rate
- Average Patient Age
- Total Patients
- Appointment Distribution

## Dashboard Pages

1. Overview
2. Patient Demographics
3. Appointment Analysis
4. No Show Insights

## Business Insights

- Identified factors influencing appointment attendance.
- Analyzed patient demographics and no-show trends.
- Tracked appointment patterns across different groups.
- Highlighted opportunities to reduce missed appointments.
