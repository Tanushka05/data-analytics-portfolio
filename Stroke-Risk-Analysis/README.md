# Stroke Risk Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes patient health data to identify major risk factors associated with stroke occurrence.

## Tools Used

- Power BI
- Power Query
- DAX
- Excel

## Key KPIs

- Total Patients
- Stroke Cases
- Average Age
- High Risk Patients
- Gender Distribution
- Risk Factor Distribution

## Dashboard Pages

1. Overview
2. Risk Factor Analysis

## Business Insights

- Identified major stroke risk indicators.
- Analyzed age and gender impact on stroke occurrence.
- Evaluated patient health conditions and risk profiles.
- Highlighted patterns useful for preventive healthcare planning.
